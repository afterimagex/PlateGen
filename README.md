# Chinese Plate Generator
# 中国车牌生成器

## 用于训练车牌数据

+ 支持多种格式车牌
+ 可以自定义车牌
+ 比较完善的数据增广
+ 支持新能源车牌
+ 支持港澳车牌
+ 支持双行车牌
